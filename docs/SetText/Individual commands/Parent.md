# Parent

Change the caller to an [Entity](../../Entities/Entity.md)'s [NPCControl](../../Entities/NPCControl/NPCControl.md).

## Syntax

````
|parent,entity|
````

## Parameters

### `entity`:  int

The [Entity id](../Common%20commands%20id%20schemes/Entity%20id.md) used to set the caller to its NPCControl. This must be a valid int value or an exception will be thrown.
