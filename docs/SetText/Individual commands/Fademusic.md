# Fademusic

Fade the music to silence with a configurable fading speed.

## Syntax

(1)

````
|fademusic|
````

(2)

````
|fademusic,fadespeed|
````

## Parameters

### `fadespeed`: float

The speed at which the music will fade to silence. This must be a valid float or an exception will be thrown. The default value is 0.025.
